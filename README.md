# Mintlify Clone (Desktop Only)

A static landing page clone of [Mintlify](https://mintlify.com/), designed specifically for desktop viewports. This project recreates the visual fidelity of the Mintlify homepage using HTML5 and CSS3, focusing on layout precision for large screens.

## üìù Project Description

This project is a pixel-perfect recreation of the Mintlify SaaS landing page. **Note: This is a non-responsive, desktop-only build.** The layout uses fixed margins and specific dimensions optimized for widescreen monitors, making it an excellent study in static layout design and SVG integration.

## ‚ú® Features

* **Desktop-Optimized Layout**: Engineered for large screens with fixed margins (`0 220px`) to maintain the exact look of the original design.
* **Intricate Vector Graphics**: Features a hard-coded, high-fidelity SVG hero illustration (`1080x657`) directly embedded in the HTML.
* **Sticky Navigation Header**: A fixed-position header with blurred backdrop effects (`backdrop-filter: blur(8px)`).
* **Custom Typography**: Utilizes Google Fonts "Inter" and "Geist Mono" to match the original branding.
* **Modern CSS Techniques**: Uses `backdrop-filter`, CSS variables, and flexbox for alignment.
